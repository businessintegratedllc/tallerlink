# Manual del dueño — TallerLink

## Para quién es
Dueño o encargado de taller mecánico, enderezado o pintura (1–8 personas).

## Pantallas

### Inicio
Resumen del día:
- Cuántos carros hay adentro
- Cuántos esperan que el cliente apruebe
- Cuántos están en proceso
- Cuántos están listos para retirar

### Vehículos
Lista de todas las órdenes (OT).
Buscá por placa o cliente. Filtrá: en taller / listos / entregados.

### Cola del taller
Tablero visual. **Arrastrá** la tarjeta del carro entre columnas:
Recibido → Diagnóstico → Espera aprobación → Espera repuestos → Mecánica → Enderezado → Pintura → Armado → Listo

Así sabés **cuál sigue** sin gritar en el patio.

### Cotizaciones
Historial de cotizaciones y presupuestos de proyecto enviados.

### Mi taller
Nombre, WhatsApp, dirección, moneda, IVA, texto al pie de las cotizaciones.

---

## Día típico

1. Llega el cliente → **+ Ingresar carro**
2. Revisan el auto → pestaña **Cotización** (o **Presupuesto proyecto** si es choque)
3. **Enviar por WhatsApp**
4. Cliente aprueba en el link
5. En la **Cola**, movés el carro a Mecánica o Pintura
6. Cuando termina → **Listo** → **Avisar por WhatsApp**
7. Al entregar → estado **Entregado**

---

## Cotización vs presupuesto proyecto

| | Cotización | Presupuesto proyecto |
|--|------------|----------------------|
| Uso | Frenos, servicio, fallas | Choque, chapa y pintura |
| Forma | Lista de ítems | Etapas (desarme, pintura…) |
| Aprobación | Link WA | Link WA |

---

## Atajos mentales

- **Bahía parada** = mirá “Sin aprobar” en Inicio  
- **¿Cuál sigue en pintura?** = Cola → columna Pintura  
- **Cliente llama “¿ya va?”** = Estado y WA → mensaje listo
